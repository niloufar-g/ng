HTML مخفف عبارت Hyper Text Markup Language می باشد. که گسترده ترین زبان استفاده شده در طراحی صفحات وب است. سه عنصر < html > و < head > و < body > با هم ساختار اصلی یک صفحه وب را تشکیل میدهند.
در ادامه مثالی از چند تگ مورد استفاده HTML را بیان می کنیم. تگ < sub > متن را به صورت زیر خط نشان میدهد مانند: زیر خط
تگ <abbr > و < acronym > برای نشان دادن عبارات کامل کلمات مختصر استفاده شده در صفحه بکار میروند. مانند PTO
